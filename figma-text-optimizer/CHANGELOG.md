## [1.0.0] - 2026-02-05

### 新增 / Added

#### 核心功能
- **文本扫描识别** - 支持扫描选中区域或整个页面的所有文本节点
- **智能文本分类** - 自动识别 8 种文本类型：按钮、标题、描述、占位符、反馈、标签、链接、通用
- **AI 优化引擎** - 根据文本类型和上下文智能优化文案
- **对比预览** - 优化前后对比展示，支持编辑后再应用
- **一键应用** - 直接将优化后的文案更新到 Figma 设计稿

#### AI 模型支持
- **OpenAI** - 支持 GPT-4o, GPT-4o Mini, GPT-4 Turbo
- **Anthropic Claude** - 支持 Claude 3.5 Sonnet, Claude 3 Opus, Claude 3 Haiku
- **Google Gemini** - 支持 Gemini 1.5 Pro, Gemini 1.5 Flash
- **兼容模型** - 支持任何 OpenAI 兼容 API（如 Ollama、DeepSeek 等）

#### 自定义 Agent
- 创建多个优化 Agent，针对不同项目/品牌
- 自定义 System Prompt，定制优化风格
- 为每个 Agent 配置专属的品牌词库和规则
- API 连接测试功能

#### 品牌词库
- 设置品牌专有名词的正确写法
- 全局词库和 Agent 专属词库
- 启用/禁用单个词条

#### 优化规则
- 添加全局优化规则
- 针对特定文本类型设置规则
- 启用/禁用单个规则

#### 历史记录
- 记录所有优化操作
- 一键还原到原始文案
- 支持清空历史

#### 导出功能
- 支持 JSON 格式导出
- 支持 CSV 格式导出
- 支持 Markdown 格式导出
- 复制到剪贴板或下载文件

#### 用户界面
- 现代化 React UI
- 支持 Figma 深色/浅色主题
- 分类筛选器
- 实时优化状态显示
