# 隐私政策 / Privacy Policy

**最后更新日期 / Last Updated**: 2026-02-05

## 中文版本

### 概述

「文本智能优化助手」（以下简称"本插件"）是一款 Figma 插件，旨在帮助用户通过 AI 技术优化界面文案。我们非常重视您的隐私，本隐私政策说明了我们如何收集、使用和保护您的信息。

### 数据收集

本插件收集和处理以下类型的数据：

#### 1. Figma 设计数据
- **文本内容**：插件会读取您在 Figma 中选择的文本节点内容
- **上下文信息**：包括文本节点的父级组件名称、位置等元数据
- **用途**：仅用于生成优化建议，不会存储或传输到我们的服务器

#### 2. 用户配置数据
- **API Key**：您输入的 AI 服务 API Key
- **Agent 配置**：您创建的优化 Agent 设置
- **品牌词库和规则**：您自定义的品牌词条和优化规则
- **存储位置**：所有配置数据仅存储在您的本地设备（通过 Figma clientStorage）
- **不会上传**：这些数据不会上传到我们的服务器

#### 3. AI 服务请求
- 当您使用优化功能时，文本内容会发送到您配置的 AI 服务提供商（如 OpenAI、Anthropic、Google）
- 这些请求直接从插件发送到 AI 服务商，不经过我们的服务器
- 请参阅相应 AI 服务商的隐私政策了解其数据处理方式

### 数据存储

- **本地存储**：所有用户数据（设置、历史记录、API Key）仅存储在您的本地设备
- **无云端同步**：本插件不提供云端同步功能，数据不会上传到任何服务器
- **无账户系统**：本插件不需要注册或登录账户

### 第三方服务

本插件可能与以下第三方 AI 服务交互（由用户主动配置）：

| 服务商 | 隐私政策链接 |
|-------|------------|
| OpenAI | https://openai.com/privacy |
| Anthropic | https://www.anthropic.com/privacy |
| Google (Gemini) | https://policies.google.com/privacy |

请注意，当您使用这些服务时，您的数据将受到相应服务商的隐私政策约束。

### 数据安全

- API Key 存储在本地，使用 Figma 提供的安全存储机制
- 所有 AI API 请求通过 HTTPS 加密传输
- 本插件不会将您的数据分享给任何第三方（AI 服务商除外）

### 您的权利

您可以随时：
- 删除存储的配置和历史记录（通过插件设置）
- 停止使用本插件
- 联系我们了解更多信息

### 儿童隐私

本插件不面向 13 岁以下的儿童，我们不会故意收集儿童的个人信息。

### 隐私政策更新

我们可能会不时更新本隐私政策。更新后的政策将在此页面发布，并注明更新日期。

### 联系我们

如果您对本隐私政策有任何疑问，请通过以下方式联系我们：
- 在 Figma 社区插件页面留言
- 在 GitHub 仓库提交 Issue

## English Version

### Overview

"Text Optimizer Assistant" (hereinafter referred to as "the Plugin") is a Figma plugin designed to help users optimize UI copy using AI technology. We take your privacy seriously, and this Privacy Policy explains how we collect, use, and protect your information.

### Data Collection

The Plugin collects and processes the following types of data:

#### 1. Figma Design Data
- **Text Content**: The plugin reads text node content you select in Figma
- **Context Information**: Including parent component names, positions, and other metadata
- **Purpose**: Used only to generate optimization suggestions, not stored or transmitted to our servers

#### 2. User Configuration Data
- **API Keys**: AI service API keys you enter
- **Agent Configurations**: Optimization Agent settings you create
- **Brand Terms and Rules**: Custom brand terms and optimization rules
- **Storage**: All configuration data is stored only on your local device (via Figma clientStorage)
- **Not Uploaded**: This data is not uploaded to our servers

#### 3. AI Service Requests
- When you use the optimization feature, text content is sent to your configured AI service provider (e.g., OpenAI, Anthropic, Google)
- These requests are sent directly from the plugin to the AI provider, not through our servers
- Please refer to the respective AI provider's privacy policy for their data handling practices

### Data Storage

- **Local Storage**: All user data (settings, history, API keys) is stored only on your local device
- **No Cloud Sync**: The plugin does not offer cloud synchronization; data is not uploaded to any server
- **No Account System**: The plugin does not require registration or login

### Third-Party Services

The Plugin may interact with the following third-party AI services (configured by the user):

| Provider | Privacy Policy Link |
|----------|-------------------|
| OpenAI | https://openai.com/privacy |
| Anthropic | https://www.anthropic.com/privacy |
| Google (Gemini) | https://policies.google.com/privacy |

Please note that when you use these services, your data will be subject to their respective privacy policies.

### Data Security

- API keys are stored locally using Figma's secure storage mechanism
- All AI API requests are encrypted via HTTPS
- The plugin does not share your data with any third party (except AI service providers)

### Your Rights

You can at any time:
- Delete stored configurations and history (via plugin settings)
- Stop using the plugin
- Contact us for more information

### Children's Privacy

The plugin is not intended for children under 13, and we do not knowingly collect personal information from children.

### Privacy Policy Updates

We may update this Privacy Policy from time to time. Updated policies will be posted on this page with the update date noted.

### Contact Us

If you have any questions about this Privacy Policy, please contact us:
- Leave a comment on the Figma Community plugin page
- Submit an Issue on the GitHub repository
